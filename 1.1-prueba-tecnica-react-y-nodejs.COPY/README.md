Ejercicio: Add y eliminar elementos de una lista (React)
Requisitos: -Tener instalado Nodejs.
-Tener instalado npm.

Duracion: 40 minutos.
Enunciado:

Crear una app en React que implemente un campo de texto y un boton para Add un elemento.
Cuando se hace click en el boton, el texto en el campo de entrada debe agregarse a continuacion en una lista de elementos.

Ademas, cada vez que se hace click en cualquier elemento de la lista, debe eliminarse de la lista.

*Dar importancia a la funcionalidad y usabilidad, mas al visual design.
*El codigo debe ser enteramente desarrollado en Typescript.
